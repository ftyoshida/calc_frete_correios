# calc_frete_correios
Calculo de frete e prazo dos Correios


Exemplo de envio dos dados da encomenda JSON via POST:

Content-Type : application/json

``` bash
{
"nCdEmpresa": "",
"sDsSenha": "",
"nCdServico": "41106",
"sCepOrigem": "04307000",
"sCepDestino": "42810280",
"nVlPeso": "10",
"nCdFormato": "1",
"nVlComprimento": "30",
"nVlAltura": "20",
"nVlLargura": "20",
"nVlDiametro": "0",
"sCdMaoPropria": "n",
"nVlValorDeclarado": "0",
"sCdAvisoRecebimento": "n"
}
```

## Codigos de Servico (nCdServico)

``` bash

Código	:	Serviço
04014	:	SEDEX à vista
04510	:	PAC à vista
04782	:	SEDEX 12 ( à vista)
04790	:	SEDEX 10 (à vista)
04804	:	SEDEX Hoje à vista

```

Fonte:
